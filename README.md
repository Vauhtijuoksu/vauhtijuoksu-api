# Vauhtijuoksu API

## Requirements
* java 16
* gradle

## Generating API
```shell
./gradlew build
```

## Testing the API
A mock server using the API specification (from `main` branch) is served at https://mockapi.dev.vauhtijuoksu.fi

## Using the API
The API server is running at https://api.dev.vauhtijuoksu.fi
